HTML Guide

  • Chat Container

    It all starts with a container div with an id of chat-container. Once, you have created this div. Your HTML will look like this:

    <div id="chat-container"></div>
    The chat container makes sure that our chat layout sticks to the bottom of the page.

  • Tabs Container

    Now, we have to make an ul (unordered list) inside our chat container with an id of tabs-container. Once you have added this ul, your HTML will look like this:

    <div id="chat-container">
        <ul id="tabs-container"></ul>
    </div>
    The ul#tabs-container (unordered list) s just a container for our tabs.

  • Understanding Tabs

    What is a tab?

    When we say tab; we are talking about something like this as a whole:

    The tab above is a chat tab but there is a special tab known as users list tab. It looks like this:

    A chat tab lets user chat with other users. There can be many chat tabs in a page.

    A users list tab shows a list of users available for chat. There will be only one such tab in a page. Do you think showing more than one will be okay? Ofcourse, not. As it will occur only once, that's why we are considering it as a special tab.

    Now, you have basic understanding what are tabs. Read below to know more about them:

    Each tab is just a li (list item) with a class of tab. Each tab can have three sections:

    1. The header
    2. The body
    3. The footer

    The sections are just a div with a class name. Like if you are going to add a header, you'll make a div with class of tab-header. Similary, a class of tab-body will be added to the div when making the body. Same goes for the footer; a tab-footer class.

    Usage of these sections depends on you. Probably, the header and the body will be present in each tab. The footer may or may not be present.

    The three sections have definite height in CSS. You are free to change it as you like. I have only used the standard heights. The users list tab's body is bigger in height as compared to other tabs.
    In order to toggle the tabs; just toggle the class active on the li.tab.

  • Users List Tab

    We will make our users list tab by making a li (list item) with a class of tab and an additional class users-list-tab in our ul#tabs-container. Our HTML will look like below now:

    <div id="chat-container">
        <ul id="tabs-container">
            <li class="tab users-list-tab"></li>
        </ul>
    </div>

    Looking into the inner HTML structure

    • The header

      We have only created the li (list item) for users list tab. Now, we have to add sections as we talked about in Understanding Tabs. The first section will be the header. We can add ANYTHING to our header as it is just a div (with the class of tab-header). For now, we will just type Chat [2] (we're showing that 2 users are online). Our HTML will look like below now:

      <div id="chat-container">
          <ul id="tabs-container">
              <li class="tab users-list-tab">
                  <div class="tab-header">Chat [2]</div>
              </li>
          </ul>
      </div>

    • The body

      We have created the header for users list tab. Now, we will make the body. In our li.tab.users-list-tab, we will add another div (with the class of tab-body). Go ahead and make the body div.

      The body will contain a LIST of users online. So, we will make a ul (unordered list) in the div.tab-body we just created. Now, we can add users as li (list items).

      In each of our list item we'll show an avatar next to the name of the person. As it requires styling so, we need some CSS but you don't have to write CSS yourself. Just add class of users-list to the ul you just created in the body div.

      Our HTML will look like below now:

      <div id="chat-container">
          <ul id="tabs-container">
              <li class="tab users-list-tab">
                  <div class="tab-header">Chat [2]</div>
                  <div class="tab-body">
                      <ul class="users-list"></ul>
                  </div>
              </li>
          </ul>
      </div>

      We'll add three dummy users now or three li (list item) in ul (unordered list) we just created. Our li (list items) will contain a p (paragraph) and an img (image). The markup for showing a user will be something like this:

      <li>
         <img class="avatar" src="<User Avatar Link>" alt="<User Avatar Link>">
         <p><Username></p>
      </li>

      I've added a class of avatar to the img so, remains at the fixed size of 32x32.

      When you'll be developing on the layout, you may pull out the users from a database and loop through markup above.

      As we had decided to make three dummy users so, we will make three li with the markup shown as above. The avatar will be same for each user. But, I decide to name those three dummy users as Tommy Vercetti, John Marsh and Trevor Phillips; my favourite video game characters. BTW, Our HTML will look like below now:

  • Chat Tabs

    Chat Tabs are also li (list items) with just the class of tab. They will contain a header, a body with messages and other cool stuff and a footer with a textarea or a form.

    Chat Tabs will created dynamically when user clicks one of the person from users list.

    How do I create a chat tab dynamically?

    Although, you have to write the JS code yourself. But here is a short guide in the steps form:

    1. You'll bind an event handler to each li in users' list.

    2. In that handler, you will define statements for creating a li with a class of tab, fetch messages from the database and adding proper HTML to the li you just created and appending it to ul#tabs-container.

    3. Don't forget to add attach event handler to the elements in the tab as well like you have to make the tab collapsible when user click the header, you have to remove the tab when user click the remove icon somewhere in the header, you have to send message when user presses enter in the textarea e.t.c.

    Please read the FYI note below That will surely bring you some relief.

    We will study the chat tabs by making one statically (we'll just add the structure in the HTML)

    Make a li with the class of tab in ul#tabs-container. Our HTML will look like this now:

    <div id="chat-container">
        <ul id="tabs-container">
            <li class="tab users-list-tab">[...]</li>
            <li class="tab"></li>
        </ul>
    </div>

    Looking into the inner HTML structure

    Now, we will look into the inner structure of a chat tab which was just created:

    • The header

      A chat tab will surely have a header as it is necessary for many things. A header will usually contain the name of the person with whom the user is chatting and a bunch of settings dropdown but we'll stick just to showing the name. Remember how do we make tab's header? A div with the class of tab-header.

      We'll put an a anchor to our header pointing to the profile link of the person with whom user is chatting and the name of the person as well. Now, our HTML will look like this:

      <div id="chat-container">
          <ul id="tabs-container">
              <li class="tab users-list-tab">[...]</li>
              <li class="tab">
                  <div class="tab-header">
                      <a href="/user/<Username>"><Fullname></a>
                  </div>
              </li>
          </ul>
      </div>

    • The body

      Now, we'll add body to our static chat tab. Go ahead and make a div with a class of tab-body. But, wait what would we add in the body? We'll add a LIST of dummy messages. As we'll make a LIST so, we need an ul (unordered list). Each message will be child of that ul which means it will be bunch of li. Let us make our body and the ul. Our HTML will look like below now:

      <div id="chat-container">
          <ul id="tabs-container">
              <li class="tab users-list-tab">[...]</li>
              <li class="tab">
                  <div class="tab-header">
                      <a href="/user/<Username>"><Fullname></a>
                  </div>
                  <div class="tab-body">
                       <ul class="messages"></ul>
                  </div>
              </li>
          </ul>
      </div>

      Now, messages need styling but you don't have to write CSS yourself; some CSS classes are already available which will make the messages pretty. Just add the class of message to the lis.

      If the message was sent by the current logged in user then add a class of me but if the message was recieved by the user then add a class of you. It is basically for differentiating the messages. The message sent or recieved can have different styles. Like by default, the me is floated towards left and the you is floated toward right.

      A message will contain img (image) avatar of the person indicating who sent that message and a div because our message may contain multiple p (paragraphs) or img (image).

      The img (image) will have a class of avatar and the div will have a class of bubble. Let us make only one message for now. Our HTML for a simple chat message will look like below:

      You may want to show when conversation started. So, before outputting the conversation make a li with a class of date in the ul.messages and add the time like Today at 4:20pm.

      Now, you may want to add some extra information about the last recieved message just as facebook does. They tell you that Seen at 5:32pm, Sent from Messenger, He is typing e.t.c. You can make a div and add any message to it just below our ul.messages. I will add the message that He is typing so, HTML will look like this now (You are not limited to adding only one div or only text inside those div; you can add ANYTHING in it but make sure it doesn't look bad :D )

  • Tab Scroller

    Now, so far we have covered users list tab and chat tabs. But there is one interesting feature you may like to implement just like facebook does. Have you ever experienced opening many chat tabs on facebook? You'll see as the width of the tabs taken together becomes greater than the width of the viewport/screen; some of chat tabs are hidden from the screen and a section appears indicating a number That number of chat tabs are hidden from the screen. I called it as Tab Scroller. A dropdown containing the name of persons whose chat tabs were hidden is shown when you click it.

    The styling of Tab Scroller comes with the css file. If you decide to implement this feature and want to write JS functions for it then you will just make a div with an id of tab-scroller and append it to our chat-container. Now, our HTML will look like below: 

    <div id="chat-container">
        <ul id="tabs-container">
            <li class="tab users-list-tab">[...]</li>
            <li class="tab">[...]</li>
        </ul>
        <div id="tab-scroller"></div>
    </div>

    It doesn't matter where the HTML of tab scroller appears. It is absolute positioned so, even if you put it before your ul#tabs. It will still work.

    • The header

      Now, we'll have a header in the tab scroller div. Remember what class do we add to make a header div? It's tab-header class. Well, tab scroller is just like tab but with small width header and different body. It shows a dropdown. Once, you have made the header; our tab scroller div will look like below:

      <div id="tab-scroller">
          <div class="tab-header">
              <b>0</b>
          </div>
      </div>

      You can see that in our header div, I have only added a number 1 just for the demo. You will change this number when you have hidden some chat tabs. It is just a div so, you can add anything. You can style the number as you like.

    • The dropdown

      Now, instead of adding a body to our tab scroller. We will add a dropdown. This dropdown will contain the names of the chat tabs which were hidden from the screen. Just for the sake of demo we'll make one. We will make an ul (unordered list) with a class of dropdown.

      Now, we'll add one li to our dropdown. In that li, we can just type in any name for the sake of demo. Add your name in it and see how does it look. I will add my name: Muhammad Talha. So, our Tab scroller will look like this now: (It is complete, BTW)

      <div id="tab-scroller">
          <div class="tab-header">1</div>
          <ul class="dropdown">
              <li>Muhammad Talha</li>
          </ul>
      </div>
      In order to show/hide the dropdown; just toggle the class active on the div#tab-scroller.
      Congratulations! We've successfully covered tab scroller.

CSS Files

I'm using three CSS files in CSS Chat Layout:

  1. normalize.min.css
  2. chat.css
  3. chat-theme.css
  5. chat-ie.css

  • normalize.min.css

    The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

  • chat.css

    This is the main css file which contains all the styles. You can easily edit the CSS file as you like. It is commented to guide you that which properties are important and why we have used them.

    To ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

  • chat-theme.css

    A classical gradient theme. It looks better than default. Plus, this file shows how you can easily customize the chat layout.

  • chat-ie.css

    This is a conditional stylesheet. It will be included if your page is loaded in old IE browsers. It basically hides the chat layout from your website instead of breaking all of the page. The old browsers do not support many properties require to make the layout.

    As this file is to be included conditionally, so its HTML looks a bit different. Like this:

    <!--[if lt IE 7]>
        <link rel="stylesheet" type="text/css" href="css/chat-ie.css">
    <![endif]-->

    The modern browsers will ignore it. But IE will read it. It will read the condition. The condition says to Internet Explorer:

    Bro, if your version is less 7 then include this file.

    If the version is less than 7 then Internet Explorer will say:

    Everything's cool, man. I will include it.

    Then IE will apply whatever styles are in the CSS file. Go ahead and open css file and you will see it has something like this in it:

    div#chat-container {
        display: none;
    }

    So, basically; The chat layout will be hidden in the IE browsers having version less than 7.

LESS File

A LESS file is also included so, you can edit the CSS Chat Layout as you like. Don't worry if you don't know about LESS; you can still edit the CSS file

There are some important variables in the LESS file. Just by editing those variables; you can control the whole layout:

// Base font size
@fontSize: 13px;

@primaryChatColor: #2574A9;
@secondaryChatColor: #EAEDF1;
@tertiaryChatColor: #D3D7DC;
@tertiaryShadeChatColor: #95A5A6;

@chatSidebarWidth: 235px;
@chatTabWidth: 215px;
@chatTabActiveWidth: 300px;

@avatarSize: 35px;

@chatTabBodyHeight: 220px;
@chatTabHeaderHeight: 35px;
@chatTabFooterHeight: 30px;
@chatUsersListTabMinHeight: 400px;
@chatUsersListTabMaxHeight: 400px;

/*
	To show these text messages you need to make sure that the list elements are completely empty.
	There should be no whitespace because :empty looks for completely empty elements.
*/
@noMessagesText: "No messages to show.";
@noUsersText: "No users to show.";

/*
	@chatMyMessageBubbleColor means the message user have sent.
	@chatTheirMessageBubbleColor means the messages which user have recieved. 
	The background colors of chat bubbles.
*/
@chatMyMessageBubbleColor: #ffffff;
@chatTheirMessageBubbleColor: #ffffff;
